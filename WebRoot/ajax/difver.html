REACTJS version (beta)
Includes SeedProject

We are very excited to announce that SmartAdmin's ReactJS (beta) version is finally released. We feel this will be a great leap for users who are trying to develop in the new trend for ReactJS. The current release also includes SeedProject along with compact bundles for you to start your project smoothly.

Technical Note! One visual flaw you will notice right away is the slight misalignment of some of the elements due to rogue span tags. This is currently a technical limitation of React; it wraps any floating text nodes in a span so that it can assign it an ID and refer back to it later. In a future version of React hopefully we can remove this restriction.

We welcome you to explore this brand new version and experience the theme hands-on!
